# Vercel AI SDK

The AI SDK is the TypeScript toolkit for building AI-powered applications and agents. It provides a unified interface for working with multiple AI providers, framework-agnostic UI hooks, and powerful tools for building sophisticated AI applications.

## Quick Start

```bash
npm install ai @ai-sdk/react @ai-sdk/openai-compatible
```

```tsx
import { generateText } from 'ai';
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';

const ollama = createOpenAICompatible({
  name: 'ollama',
  baseURL: 'http://localhost:11434/v1',
});

const { text } = await generateText({
  model: ollama('llama3.2'),
  prompt: 'What is the meaning of life?',
});
```

## Key Features

### Unified Provider Interface

Switch between AI providers with just a few lines of code:

```ts
// Ollama (local)
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
const ollama = createOpenAICompatible({
  name: 'ollama',
  baseURL: 'http://localhost:11434/v1',
});
const model = ollama('llama3.2');

// Groq (cloud)
import { createGroq } from '@ai-sdk/groq';
const groq = createGroq({ apiKey: process.env.GROQ_API_KEY });
const model = groq('llama-3.2-90b-text-preview');

// Together.ai
import { createTogether } from '@ai-sdk/together';
const together = createTogether({ apiKey: process.env.TOGETHER_API_KEY });
const model = together('meta-llama/Llama-3.2-90B-Vision-Instruct-Turbo');
```

### Streaming Support

Built-in streaming for real-time responses:

```tsx
import { useChat } from '@ai-sdk/react';

export default function Chat() {
  const { messages, input, handleSubmit, handleInputChange } = useChat();
  // Automatic streaming UI
}
```

### Structured Data

Generate type-safe JSON objects:

```ts
import { generateObject } from 'ai';
import { z } from 'zod';

const { object } = await generateObject({
  model: ollama('llama3.2'),
  schema: z.object({
    title: z.string(),
    items: z.array(z.string()),
  }),
  prompt: 'Generate a todo list',
});
```

### Tool Calling

Enable AI to use external tools:

```ts
import { generateText, tool } from 'ai';

const { text } = await generateText({
  model: ollama('llama3.2'),
  tools: {
    getWeather: tool({
      description: 'Get weather',
      parameters: z.object({ location: z.string() }),
      execute: async ({ location }) => {
        return { temp: 72, condition: 'sunny' };
      },
    }),
  },
  prompt: 'What is the weather in SF?',
});
```

## Supported Frameworks

- **React / Next.js** - Full support with `@ai-sdk/react`
- **Vue / Nuxt** - Full support with `@ai-sdk/vue`
- **Svelte / SvelteKit** - Full support with `@ai-sdk/svelte`
- **Solid** - Full support with `@ai-sdk/solid`
- **Node.js** - Backend support with AI SDK Core

## Example Use Cases

- **Chatbots**: Build conversational interfaces with streaming
- **Content Generation**: Automate writing and creative tasks
- **Data Extraction**: Extract structured information from text
- **AI Agents**: Create autonomous systems with tool usage
- **RAG Systems**: Build knowledge-based question answering
- **Multi-Modal Apps**: Process text, images, PDFs, and audio
